Displays the **EC2 Instance Connect Endpoint (`three-tier-eic-endpoint`)** configured within the Amazon VPC, enabling secure administrative access to the private EC2 application server without assigning a public IP address.

### Key Highlights

- Provides secure SSH connectivity to the private EC2 instance without requiring a bastion host or public IP.
- Deployed inside the custom VPC and associated with a dedicated security group for controlled administrative access.
- Enhances security by keeping the application server isolated while enabling secure management through AWS EC2 Instance Connect.
