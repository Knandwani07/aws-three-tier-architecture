Displays the **public and private subnets** created across two Availability Zones within the custom Amazon VPC. The public subnets host the **Application Load Balancer (ALB)**, while the private subnets securely host the **Amazon EC2 application server** and **Amazon RDS MySQL database**, ensuring network isolation and a production-style three-tier architecture.
