Displays the **Amazon RDS DB Subnet Group (`three-tier-db-subnet-group`)** configured with private subnets across two Availability Zones, providing secure network placement for the Amazon RDS MySQL database.

### Key Highlights

- Includes private subnets in **two Availability Zones** to support high availability and secure database deployment.
- Ensures the Amazon RDS instance is deployed within private subnets, preventing direct internet access.
- Provides dedicated network isolation for the database tier while enabling secure communication with the EC2 application server.
